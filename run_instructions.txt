# Running the App in Different Environments

## Development:
on the home laptop:
```bash
MACHINE_ENV=home NODE_ENV=development node app.js

on travel laptop:
```bash
MACHINE_ENV=travel NODE_ENV=development node app.js

## Production: 
home laptop:
```bash
MACHINE_ENV=home NODE_ENV=production node app.js

travel laptop:
```bash
MACHINE_ENV=travel NODE_ENV=production node app.js
